# みぷりん 三面図・四方向設計

## 必要方向

Unity 2Dの見下ろし型アクションRPGで使うため、最低限以下を用意する。

1. Front：正面
2. Back：背面
3. Left：左向き
4. Right：右向き

## 正面

- 顔と衣装が最も見える基本デザイン
- 武器を右手に持つ
- はちみつポットを左手または腰に下げる
- 羽は左右に広げる
- 触角は左右対称
- 表情は明るい笑顔

## 背面

- 金髪ボブの後頭部を見せる
- 羽を大きく見せる
- 背中側に蜂柄の胴体ラインを見せる
- スカートと靴を正面と同じ形状で整合
- ポットやバッグの位置を正面と矛盾させない

## 左向き / 右向き

- 横顔は幼い丸顔
- 羽は奥行きが出るように前後で重ねる
- 武器は進行方向側に出す
- 歩行時に髪・スカート・羽が少し揺れる想定

## Unityでの利用

最初は4方向フレームアニメ。
将来的にはパーツ分割＋2Dボーンアニメに移行可能。
