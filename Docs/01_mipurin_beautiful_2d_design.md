# 美麗2Dみぷりん デザイン仕様書

## 採用方針

採用案：高品質SD・アニメ塗り版

ドット絵版の特徴を残しつつ、Unity 2Dで見栄えする高解像度キャラクターへ再設計する。

## 参照元の特徴

現行みぷりんの特徴：

- 金髪ボブ
- 茶色い大きめの目
- 蜂の触角
- 茶色い獣耳風パーツ
- 透明な水色の羽
- 黄色と黒茶の蜂柄トップス
- 茶色いスカート
- 白い靴下と茶色い靴
- 小さなはちみつポット
- 正面・背面・左右向きの4方向キャラ

## 美麗2D版で残すもの

- 金髪ボブ
- 丸く幼いSD頭身
- 蜂モチーフ
- 透明な羽
- 黄色・黒茶・白・はちみつ色の配色
- はちみつポット
- 小さな冒険者感

## 美麗2D版で強化するもの

- 顔の可愛さ
- 目のハイライト
- 髪のツヤ
- 羽の透明感
- 衣装の立体感
- 武器の存在感
- Unity上で見やすいシルエット

## 画風

- 高品質SDアニメ塗り
- 明るい色
- 太すぎない輪郭線
- ゲーム画面で見やすいコントラスト
- かわいいが幼すぎない
- 花の国・蜂の王国に合うファンタジー感

## 基本装備

- 右手：蜂の針型ショートスピア
- 左手または腰：はちみつポット
- 背中：透明な4枚羽
- 小物：肩掛けバッグ、はちみつチャーム

## NG

- リアルな虫っぽさを強めすぎない
- セクシー寄りにしない
- 頭身を上げすぎない
- 蜂要素を消さない
- ドット絵のまま拡大しただけにしない
