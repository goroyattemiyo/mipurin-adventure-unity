# 09 Final Game System

## 目的

Unity版『ミプリンの冒険』の最終的なゲームシステム像を固定する。

このドキュメントは、素材制作・Unity実装・ゲーム仕様設計が分かれても、最終的なゲームの方向性がブレないようにするためのもの。

## 最終ジャンル

```text
美麗2Dみぷりんの部屋制アクションローグライトRPG
```

## 参考にするもの

ゲームシステムの骨格として、DunHero系の部屋制アクションローグライト構造を参考にする。

ただし参考にするのは、以下のシステム部分に限定する。

- 小部屋単位で進行する
- 部屋に入ると敵が出る
- 敵を倒すと部屋クリア
- クリア後に報酬・強化を選ぶ
- 複数部屋を進む
- 一定数進むとボス
- 1プレイごとにビルドが変わる
- 負けたら最初から
- 一部成長や解放は残る

見た目・世界観・キャラクター・敵デザイン・演出は、ミプリン独自の蜂・花・はちみつファンタジーとして作る。

## 基本ゲームループ

```text
拠点
↓
ステージに出発
↓
小部屋に入る
↓
敵を倒す
↓
報酬・祝福・アイテムを得る
↓
次の部屋を選ぶ
↓
複数部屋を進む
↓
ボスを倒す
↓
クリア or ダウン
↓
持ち帰った素材で拠点強化・図鑑登録・装備解放
```

## 部屋タイプ

最終的には以下の部屋タイプを用意する。

```text
戦闘部屋
報酬部屋
回復部屋
ショップ部屋
イベント部屋
ボス部屋
```

最初に実装するのは戦闘部屋のみ。

## 戦闘システム

みぷりんは軽快な近距離アクションを基本にする。

### 基本攻撃

```text
針型スピアによる近距離攻撃
```

### 将来のサブ攻撃案

```text
はちみつ弾
花粉ショット
羽ばたき衝撃波
花びら弾
```

### 回避・移動

```text
短いふわっとダッシュ
羽ばたき移動
ノックバック制御
```

## 報酬・祝福

部屋クリア後に、複数の報酬候補から選ぶ方式を目指す。

例：

```text
攻撃力アップ
攻撃速度アップ
移動速度アップ
最大HPアップ
HP回復
はちみつ弾追加
花びら弾追加
羽ばたき強化
```

## 成長要素

### 1プレイ中の成長

- 武器強化
- 祝福
- 一時ステータスアップ
- 回復アイテム
- 特殊効果つきはちみつ

### 永続成長

- 図鑑登録
- 装備解放
- 初期HP微増
- 初期武器解放
- 祝福アンロック
- 拠点強化

## 拠点

将来的に、小さな蜂の拠点を作る。

候補：

```text
みぷりんの家
花壇
はちみつ工房
武器置き場
図鑑
おともキャラ
出発ゲート
```

拠点は、永続成長・装備変更・図鑑確認・出発準備の場所にする。

## 開発フェーズ

### Phase 1: CharacterTest完成

- みぷりんが動く
- 攻撃できる
- 敵を倒せる
- HPが減る
- ダウンする
- 簡易ステージで遊べる

### Phase 2: 1部屋ゲーム化

- 床・壁・枠
- 敵複数
- クリア判定
- リスタート
- 簡易UI

### Phase 3: ミニダンジョン化

- 3〜5部屋
- 敵ランダム
- 報酬部屋
- 回復部屋
- 小ボス

### Phase 4: ローグライト化

- 武器
- 祝福
- ドロップ
- 部屋生成
- ステージ進行
- ゲームオーバー
- リザルト

### Phase 5: 拠点と永続成長

- 花壇
- 図鑑
- 素材持ち帰り
- 強化
- 解放要素

### Phase 6: デモ版

- 1エリア
- 通常敵3〜5種
- ボス1体
- 武器3種
- 祝福10種前後
- プレイ時間10〜20分

## 注意

最終系は大きくしすぎない。

今の最優先は、CharacterTestを遊べる1部屋にすること。

実装順は以下を守る。

```text
1. CharacterTestを遊べる1部屋にする
2. 敵1体を正式敵に差し替える
3. 1部屋クリア判定を作る
4. 報酬を1つ出す
5. 2〜3部屋つなげる
6. 簡易ボスを出す
```
