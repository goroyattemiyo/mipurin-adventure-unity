# Mipurin Adventure Unity Starter

Unity 2D版「ミプリンの冒険」の初期設計リポジトリです。

## 方針

- 現行Web版を置き換えず、Unity 2Dリメイク版として別ラインで開発する
- ドット絵版みぷりんを卒業し、美麗2DのSDアニメ塗りキャラとして再設計する
- まずは「みぷりんがUnity上で動く」ことを最優先にする

## 最初のMVP

1. CharacterTest シーン
2. みぷりんの移動
3. Idle / Walk / Attack / Hurt / Dead
4. 敵1体
5. 攻撃判定
6. HPとゲームオーバー
7. 1部屋クリア

## デザイン方針

採用案は、チャットで選んだ「2つ目」の高品質SD・アニメ塗り版です。
